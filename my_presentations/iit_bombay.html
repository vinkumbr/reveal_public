Given locations \(\mathbb{X}\) and communities \( \sigma\)

\[A_{uv}=1 \begin{cases}\text{w.p. } f_{in }(X_u,X_v) & \text{if } \sigma(u)=\sigma(v)\\ \text{w.p. } f_{out}(X_u,X_v) & \text{if } \sigma(u) \neq \sigma(v) \end{cases}\]

\(\mathbf{A}\) \(\sim GKBM(\lambda,n,f_{in},f_{out})\)

Problem Formulation

\(\mathbf{A}\) \(\sim GKBM(\lambda,n,p,q,\phi)\)

Problem: Given the locations \(\mathbb{X}\) and the graph \(\mathbf{A}\), recover \( \sigma_n\) exactly.

An estimate \( \hat{\sigma}_n\) of \( \sigma_n\) is said to recover the communities exactly if \[\lim_{n\to \infty}\mathbb{P}\big( \color{#a689e0}{\hat{\sigma}_n} \in\{\pm \color{#a689e0}{\sigma_n} \}\big) = 1 \]

Main results

Define \(\kappa = \sup_{x \in \Gamma}x, \ \ 0<\kappa<\infty\) and \[I_{\phi}(p,q) := 2\int_{\mathbb{R}_+}\big[1-\sqrt{pq}\phi(x)-\sqrt{(1-p\phi(x))(1-q\phi(x))}\big]dx\] Converse: If \(\lambda \kappa<1\) or \(\lambda I_{\phi}(p,q) < 1\), then exact recovery is not possible using any algorithm.
Achievability: If \(\lambda \kappa >1\) and \(\lambda I_{\phi}(p,q) > 1\), then there exists a linear time algorithm (in the number of edges) achieving exact recovery.

Impossibility: Idea

If \(\lambda \kappa<1\) or \(\lambda I_{\phi}(p,q) < 1\), then exact recovery is not possible. \[I_{\phi}(p,q) := 2\int_{\mathbb{R}_+}\big[1-\sqrt{pq}\phi(x)-\sqrt{(1-p\phi(x))(1-q\phi(x))}\big]dx\]
  • Genie-based estimator: Log-likelihood
    • \[\sum_{\small{\stackrel{v\sim 0}{\sigma_v=\sigma_0}}} \log \ (p\phi_{v0}) + \sum_{\small{\stackrel{v\sim 0}{ \sigma_v\neq\sigma_0}}} \log \ (q\phi_{v0}) + \sum_{\small{\stackrel{v\nsim 0}{\sigma_v=\sigma_0}}} \log\ (1-p\phi_{v0}) + \sum_{\small{\stackrel{v\nsim 0}{\sigma_v\neq\sigma_0}}} \log\ (1-q\phi_{v0})\]

Achievability

Q. How to recover the communities exactly when \(\lambda I_{\phi}(p,q) < 1\)?

Two phase algorithm:

Recall \(\color{#FF7F50}{\kappa}\): maximum interaction distance

Phase-I: Almost-exact recovery

  • Divide into blocks of size \(\frac{\color{#FF7F50}{\kappa}}{2}\frac{\log n}{n}\)
  • Recover exactly in an initial block
  • Propagate from a recovered block to adjacent block and so on


Phase-II: Refinement step

  • Genie-based correction step \(\qquad \qquad \qquad\)

Recovering the initial block

  • Dense graph within the block.
  • Off-the-shelf algorithms for e.g., spectral.
  • Choose \(u_0 \in V_1\) and set \(\hat{\sigma}(u_0) = +1.\)
  • Cluster using number of common neighbours of \(u\) and \(u_0\)


Lemma: For any \(p > q\) and \(\Delta > 0\), communities of nodes in the initial block \(B_1\) are recovered w.h.p., i.e., \[\color{#C8A2C8}{\mathbb{P}\Big(\bigcap_{u\in V_1} \{ \hat{\sigma}(u) = \sigma(u) \}\Big) \ge 1-\Delta n^{-c_1}\log n}\]

Label propagation

  • Assume that the estimated communities in block \(B_i\) are the true communities.
  • Evaluate the likelihood of edges to \(u \in B_{i+1}\) from vertices in \(B_i\) \[\text{sign}(\sum_{v\in V_i} \hat{\sigma}(v) \mathcal{L}(A; \hat{\sigma}))\]


Lemma: There exists an \(M \equiv M(p, q, \phi) > 0\) such that the event \[\mathcal{A_i} := \{\text{at most }M \text{ mistakes within block }B_i \}\] satisfies \( \color{#C8A2C8}{\mathbb{P}\big( \mathcal{A_{i+1}}\big| \mathcal{A_i} \big) \ge 1-c_2 n^{-9/8}}.\)

Label propagation: Main idea

Let \(\mathcal{A_i} = \{\text{at most }M \text{ mistakes within block }B_i \}\) and \(b = \# \text{ of blocks}= \frac{2n}{\kappa \log n}\).
\[\mathbb{P}\Big(\bigcap_{i=1}^{b }\mathcal{A}_i\Big) = \mathbb{P}(\mathcal{A}_1)\prod_{i=2}^{b}\mathbb{P}\Big(\mathcal{A}_i \Big| \bigcap_{j< i} \mathcal{A}_{j}\Big)\]
\[\mathbb{P}\Big(\bigcap_{i=1}^{b }\mathcal{A}_i\Big) = \mathbb{P}(\mathcal{A}_1) \cdot \prod_{i=2}^{b}\mathbb{P}\Big(\mathcal{A}_i \Big| \mathcal{A}_{i-1}\Big)\]
\(\qquad \qquad \qquad \qquad \qquad \hspace{0.5cm} \ge (1-\Delta n^{-c_1}\log n ) \cdot (1-c_2 n^{-9/8})^{\frac{2n}{\kappa \log n}} \)
\[\qquad \qquad \qquad \qquad \qquad \ge (1-\Delta n^{-c_1}\log n ) \cdot (1-\frac{2 c_2}{n^{1/8} \kappa \log n} )\] Sacrifice on probability but ensure at most \(M\) mistakes within each block.
Lemma: Fix any \(\eta >0 \). For \(A \sim \text{GKBM}(\lambda, n , p, q, \phi)\), we have that satisfies \[ \color{#C8A2C8}{\mathbb{P}\bigg( \text{Total }\# \text{ of mistakes} \le \frac{\eta n}{3\kappa}\bigg) = 1-o(1)}.\]
